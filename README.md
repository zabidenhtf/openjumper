# Open Jumper - 3D game about jumpstyle dancing

## Description
Open Jumper - 3D game about jumpstyle dancing

## Authors
ZABIDEN - main developer of game

## Controls
Space - Jump
A - Kick left
D - Kick right
W - Shuffle forward
S - Shuffle backward
E - Flip
